STORYTELLING WEB MAP

Developing an Engaging Web Solution for The Liberty Project's Bike Fundraiser

Using a combination of HTML, JavaScript, and CSS, Vanritto successfully created a scrolling GIS web map which tells the story of The Liberty Project Fundraiser. The series of chapters applied to the web map in the config.js file results in smooth transitioning between the stops. This allows the user to scroll effectively from one stop to another while providing information about the fundraiser, Mozambique, and The Liberty Project in the form of images and text.

The team was able to implement full customization for the route, icons and labels using Layers within Mapbox. This allowed for the placing of Icons upon the route line by using sort-key parameters within the HTML file within the add.Layer for these elements. Creating a dataset and Layer in Mapbox Studio for custom labels has resulted in labels being positioned where they are needed by the Icons and allowed for filtering of existing labels.
