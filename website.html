The Custom Event Webpage: Ride 4 Freedom

The Client's finished event webpage needed to be easy-to-use, user-friendly, interactive, and engaging in order to tell the story of the work The Liberty Project does in Mozambique, as well as encourage donations and the public to sign up and ride alongside one another for the fundraiser.

Developing an Engaging Web Solution for The Liberty Project's Bike Fundraiser

The event webpage showcases a simplistic design that follows The Liberty Project’s branding and predetermined styling choices, with plenty of buttons for users to be able to donate and register to the event. The event website makes use of third-party applications Donorbox, Eventbrite, and My Virtual Mission. These third-party applications will take the user off the main event page, to access the specific event related content on those applications.The event webpage is divided into five main sections:

Introduction | Donation | The Virtual Ride | Registration | Instagram

The sections focus on donations and the embedded Donorbox widget; and the scrollytelling GIS web map and the virtual ride. Users are able to access the virtual event by selecting the My Virtual Mission button. A new tab will open a new window to the My Virtual Mission map showing where registrants are along the bike route. The registration section is simple, and encourages the user to get involved through the use of images and easy to use donation or event registration buttons. FInally, the Instagram links directly to The Liberty Project’s Instagram account. This widget shows both live updates of the event, and when a user clicks on any of the photos, they will be brought to The Liberty Project’s Instagram profile.
